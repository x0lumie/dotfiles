# What is used?
I currently use:
  - Hyprland (WM)
  - QuickShell w/noctalia cfgs (Status Bar)
  - Rofi (App Launcher)
  - HyprShot (Screen Shotting)
  - Kitty (Terminal)
  - Oh-My-Zsh (Zsh Theming)
# Installation
Arch:

  ```
  sudo pacman -Sy hyprland quickshell rofi kitty && yay -Sy hyprshot

  ```
After running that command and everything finishes drag the config folder into ~/.config/
