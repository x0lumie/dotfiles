## What is used?
I currently use:
  - Hyprland (WM)
  - QuickShell w/noctalia cfgs (Status Bar)
  - Fuzzel (App Launcher)
  - HyprShot (Screen Shotting)
  - Kitty (Terminal)
  - Oh-My-Zsh (Zsh Theming)
##
## Installation
### Arch:

  ```
  yay -Syu hyprland quickshell fuzzel kitty hyprshot noctalia-shell
  ```
After everything installs run this to automatically clone the repo and put it in your ~/.config folder.

```
git clone https://github.com/x0lumie/dotfiles && cd dotfiles && cp -r config/* ~/.config/
```
